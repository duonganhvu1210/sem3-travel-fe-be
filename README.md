# Karnel Travels

A full-stack travel booking web application developed as an academic project. The system allows users to browse destinations, manage bookings, and provides an admin interface for managing travel services.

## Features

- User authentication
- Browse travel destinations
- Tour booking system
- Booking management
- Responsive user interface
- Admin management dashboard
- RESTful API integration

## Tech Stack

### Frontend
- React
- JavaScript
- HTML5
- CSS3

### Backend
- Spring Boot (hoặc PHP nếu backend của bạn dùng PHP)
- REST API

### Database
- MySQL

## Installation

### Clone repository

```bash
git clone https://github.com/duonganhvu1210/sem3-travel-fe-be.git
```

### Install frontend

```bash
npm install
npm run dev
```

### Configure backend

- Configure database connection
- Import database
- Run backend server
